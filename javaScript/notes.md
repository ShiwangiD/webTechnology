### History of JavaScript :=====

1995: JavaScript was created by Brendan Eich at Netscape in just 10 days. It was first called Mocha, then LiveScript, and finally JavaScript.
1996–1997: Netscape submitted it to ECMA International, which standardized it as ECMAScript.
2000s: JavaScript became essential for making web pages interactive (forms, animations, dynamic content).
2009: Node.js was released, allowing JavaScript to run outside browsers (on servers).
2015: A major update called ES6 (ECMAScript 2015) added modern features like classes, arrow functions, and modules.
Today: JavaScript is one of the most widely used programming languages for web, mobile apps, and even backend development.
